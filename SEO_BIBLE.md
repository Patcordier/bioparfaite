## CHECKLIST AVANT CHAQUE PUBLICATION

### Page / Article
- [ ] Title tag < 60 caractères, mot-clé principal inclus
- [ ] Meta description 150-160 caractères, avec CTA
- [ ] URL propre et descriptive (pas de dates ni paramètres)
- [ ] Un seul H1 par page, avec mot-clé
- [ ] Hiérarchie H1 → H2 → H3 sans sauter de niveau
- [ ] Canonical URL définie
- [ ] Open Graph (og:title, og:description, og:image)
- [ ] Twitter Card (twitter:card, twitter:title, twitter:image)
- [ ] Schema.org JSON-LD (Article, FAQPage, BreadcrumbList selon le cas)
- [ ] Images : alt descriptif, width/height, loading="lazy", format compressé
- [ ] Maillage interne : ≥3 liens vers d'autres pages du site
- [ ] Maillage interne inverse : mettre à jour les pages existantes pour lier vers la nouvelle
- [ ] Sitemap.xml mis à jour
- [ ] robots.txt vérifié (pas de blocage accidentel)

### Contenu (E-E-A-T + GEO 2026)
- [ ] Contenu original, pas de copié-collé
- [ ] Réponses directes aux questions (titres H2/H3 sous forme de questions)
- [ ] Données chiffrées, tableaux, listes numérotées (pour AI Overviews)
- [ ] Expertise démontrée (exemples concrets, expérience terrain)
- [ ] CTA vers le service payant (naturel, pas agressif)
- [ ] Longueur suffisante (1500+ mots pour articles piliers)

## PILIERS SEO 2026 (classés par impact)

### 1. CONTENU (Impact: ★★★★★)
- Publier 1 article/semaine minimum
- Cibler des mots-clés long-tail en français
- Structurer pour GEO : claims clairs, stats, tableaux, Q&A
- Contenu humain > contenu IA (Google récompense l'authenticité E-E-A-T)

### 2. BACKLINKS (Impact: ★★★★★)
- Technique Skyscraper de Backlinko : créer du contenu meilleur que les concurrents
- Articles invités sur blogs rencontre/lifestyle FR
- Forums francophones (Reddit r/france, JeuxVideo.com, forums rencontre)
- Annuaires de qualité

### 3. MAILLAGE INTERNE (Impact: ★★★★☆)
- Chaque page doit avoir ≥3 liens internes
- Ancres descriptives (pas "cliquez ici")
- Structure pyramidale : Accueil → Articles → Formulaire
- Vérifier les pages orphelines (0 lien entrant)

### 4. TECHNIQUE (Impact: ★★★☆☆)
- HTTPS actif
- Core Web Vitals optimisés (sites statiques = avantage naturel)
- Mobile-first
- Sitemap + robots.txt corrects
- Google Search Console + Bing Webmaster Tools actifs

### 5. UX / ENGAGEMENT (Impact: ★★★☆☆)
- Temps de lecture élevé (contenu captivant)
- Faible taux de rebond (maillage interne aide)
- Design mobile impeccable
- CTA clairs sans être intrusifs
